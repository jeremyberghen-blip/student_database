# Student Database

## Overview
A console-based application for keeping track of students, the classes they are enrolled in, and their grades. Built as a learning project in Java to practice object-oriented programming and basic data management.

## Features
- Add new students with their personal information
- Enroll students in classes
- Record and update grades for students
- View student information and grades in the console

## Technologies Used
- Java (SDK version 25)
- Console input/output

## What I Learned
- How to structure a simple database-like application using Java classes and objects
- Using collections to store and manage data
- Reading user input from the console and performing validation
- Implementing basic CRUD operations (Create, Read, Update, Delete)

## How to Run
1. Open the project in IntelliJ IDEA (or your preferred IDE)
2. Compile and run `Main.java`
3. Follow the console prompts to manage students and grades
